# swiftcrossUI-jamf-smartcard-tool
Cross platform swift app using SwiftCrossUI to work on macOS, Windows, and Linux. 

Tool to interact with the jamf api. Utility to easily swap an extension attribute in jamf to enable or disable smartcard enforcement or read some basic information from Jamf Pro.

Requires Stack Bundler to build [https://github.com/stackotter/](https://github.com/stackotter/swift-bundler)
